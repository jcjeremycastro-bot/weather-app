# Weather App

Check the current weather on any city of France.

## Features

1. Configurable city via external json

2. Minimal application and API request

3. Temperatures, humidity and visibility

4. Wind speed and direction

5. Sunrise and sunset times

6. Error handling and loading info

## Installation

Need Node 16 - https://www.nvmnode.com/

1. `nvm use 16`

2. `git clone https://github.com/jcjeremycastro-bot/weather-app.git`

3. `cd weather-app`

4. `npm install`

5. `npm run dev`
